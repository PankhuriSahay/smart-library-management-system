# Smart Library Management System

## Overview
Smart Library Management System is a Java-based application developed to automate and simplify library operations such as book management, issuing books, returning books, and maintaining student records. The system integrates Java with MySQL using JDBC for efficient database connectivity and record management.

## Features
- Add and manage books
- Issue and return books
- Search book records
- Maintain student details
- Database integration using JDBC
- Efficient record handling system

## Tech Stack
- Java
- JDBC
- MySQL
- OOPs Concepts

## Project Objective
The objective of this project is to digitize traditional library operations and provide an efficient platform for managing books, users, and transaction records with improved accuracy and reduced manual effort.

## Modules Used
- Book Management
- Student/User Management
- Database Connectivity
- CRUD Operations
- Exception Handling
